Welcome to eCommerce

A insecure website for placing orders, posting reviews, and communication between vendors and customers.

Get started today!

Features

  • Passwords stored in plain text.
  • No support for secure HTTPS.
  • Vulnerable to reflected and stored XSS attacks.
  • Open to simple SQL injection attacks.
  • No protection against CSRF attacks.
  • 100% trust in client requests.

Features for vendors

  • Post product offerings
  • Receive and review orders.
  • Receive valuable customer feedback.
  • Communicate with customers directly.

Features for customers

  • Search for products from different vendors.
  • Order products.
  • Post reviews for bought products.
  • Communicate with vendors directly.
