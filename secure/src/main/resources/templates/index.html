Welcome to eCommerce

A semi-secure website for placing orders, posting reviews, and communication between vendors and customers.

Get started today!

Features for vendors

  • Post product offerings
  • Receive and review orders (digitally signed using RSA cryptography).
  • Receive valuable customer feedback.
  • Communicate with customers directly (using DES encrypted messaging).

Features for customers

  • Search for products from different vendors.
  • Order products (digitally signed using RSA cryptography).
  • Post reviews for bought products.
  • Communicate with vendors directly (using DES encrypted messaging).
